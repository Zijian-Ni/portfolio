# 🌌 Zijian Ni — Portfolio

> **AI World Architect · Building the Aurora Universe**

Personal portfolio website for Zijian Ni (倪子健) — CS & AI student at the University of Leeds.

## ✨ Features

- 🎨 Deep dark tech theme with purple + cyan accents
- ✨ Custom particle/starfield canvas animation (no libraries)
- 📱 Fully responsive (mobile / tablet / desktop)
- 🌐 Chinese / English bilingual toggle
- 🎭 Scroll-triggered animations (Intersection Observer)
- 🖱️ Cursor glow follow effect
- 🃏 Flip-on-hover project cards
- 📊 GitHub stats integration
- 🌌 Aurora Universe timeline with constellation visualization
- ⚡ Pure HTML + CSS + JS — zero build tools, zero frameworks

## 🛠️ Tech Stack

- HTML5 (semantic)
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Canvas API, Intersection Observer, ES6+)
- Google Fonts (Inter, JetBrains Mono, Noto Sans SC)
- Font Awesome 6
